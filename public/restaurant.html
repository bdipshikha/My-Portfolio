Restaurant app

A single page web application designed to help the restaurant owners to maintain dishes, menu categories and prices. Seeded the SQLite database with the dish names, category names, pictures and prices of dishes dynamically using Vanilla JavaScript on Node.js. Displayed the menu categories, dishes under the categories using AJAX.
