E-Commerce app

A multi page E-commerce app designed to help jewelry merchants to maintain business using Express.JS framework with SQlite3 database, boostrap and pure css.

Admin: An admin will be able to create, update and delete categories as well as items and maintain the inventory.

User: An user will be able to browse the categories and posts. User can add item(s) to shooping cart, proceed to checkout and submit order.

Stripe credit card processing was used for the project.

https://github.com/bdipshikha/E-commerce
